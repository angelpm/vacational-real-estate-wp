Wordpress plugin for manage sells and rents real estates.  Create a property post type with all the features. can generate season prices and booking calendar.
There are two services, saleand rent. You can create differents prices season for each property. Manage the calendar booking for each property.Create two internal pages, one for listing the properties
with a search form. Another page for display single property with a contact form and a booking form.


== Installation ==
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can config settings in the setting submeno from Real Estate menu option
4. insert your Api google for address map for each property
